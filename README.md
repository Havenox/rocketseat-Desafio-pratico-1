# Desafio Pratico 1

Repositorio contendo a resolucao dos tres exercicios propostos.

## Estrutura

- **ExercicioUm**  
  Le um nome e exibe uma mensagem de boas vindas.

- **ExercicioDois**  
  Le nome e sobrenome e exibe o nome completo.

- **ExercicioTres**  
  Le dois valores double digitados pelo usuario e exibe:  
  soma, subtracao, multiplicacao, divisao com verificacao de zero e media.

## Como executar

1. Clonar o repositorio  
2. Abrir o projeto em um ambiente com .NET instalado  
3. Executar com  
```
dotnet run
```

## Tecnologias

- .NET  
- C#

## Observacao

Cada exercicio possui sua classe propria e utiliza metodos de utilidade para limpeza de tela e pausa.
