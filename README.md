# Desafio Pratico 1

Repositorio contendo a resolucao dos seis exercicios propostos no primeiro desafio pratico do modulo.

## Estrutura

- **ExercicioUm**  
  Solicita um nome ao usuario e exibe uma mensagem personalizada de boas vindas.

- **ExercicioDois**  
  Le nome e sobrenome digitados pelo usuario e exibe o nome completo concatenado.

- **ExercicioTres**  
  Permite inserir dois numeros do tipo double e exibe:  
  soma, subtracao, multiplicacao, divisao (com verificacao de zero) e media.

- **ExercicioQuatro**  
  Le uma ou mais palavras digitadas pelo usuario e conta quantos caracteres existem desconsiderando os espacos.

- **ExercicioCinco**  
  Solicita uma placa de veiculo e valida se esta no padrao brasileiro antigo (pre Mercosul):  
  3 letras seguidas de 4 numeros.

- **ExercicioSeis**  
  Exibe a data e hora atual em diferentes formatos conforme escolha do usuario:  
  formato completo, somente data, somente hora ou data com mes por extenso.

## Como executar

1. Clonar o repositorio  
2. Abrir o projeto em um ambiente com .NET instalado  
3. Executar com:

```
dotnet run
```


## Tecnologias

- .NET  
- C#

## Observacao

Cada exercicio possui sua propria classe, utiliza metodos de utilidade para limpar a tela e aguardar interacao, e segue exatamente as instrucoes solicitadas no desafio pratico.
